# desafio-react
Desafio conhecimentos básicos em React.

 O desafio consiste em montar uma página com uma listagem de Pokémons consumindo uma API já existente. O desafio deve ser feito obrigatoriamente utilizando as bibliotecas React e tailwind CSS, mas listarei abaixo outras bibliotecas que usamos por padrão, que podem ser utilizadas e serão vistas como um ponto positivo.

**O que será avaliado:**

 - uso correto de hooks do React
 - componentização
 - fidelidade ao layout original
 - organização e boas práticas de código
 - organização / estrutura de arquios e pastas do projeto
 - uso das bibliotecas obrigatórias e opcionais listadas abaixo
 - commits semanticos
 - responsividade do layout (opcional)

**O que deve ser utilizado:**

 - React 
 
 - Tailwind CSS
 
 - Axios (opcional)
 
 - React Query (TanStack Query) (opcional)
 
 - Next JS (opcional) //O Next JS é um framework criado em cima do React, então é necessário ter conhecimento prévio em React. Mesmo que não seja utilizado no projeto de desafio é altamente recomendado o estudo, por ser padrão na empresa.
